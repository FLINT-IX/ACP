ACP stands for Ace content ported. Thank you mac and golem and arend for letting me port these handful of ace missiles over to acf.

Missiles are balanced to try and fit into existing ACF missiles but while trying to keep their own identity.
Less balanced missiles can be found under 'Tactical Weapons'. 
By the nature of these weapons they never will be truely balanced, server owners who wish not to deal with these less fair missiles can set acfm_tac to 0.

I dont know how github works discord is best for balancing/bugs/etc.
