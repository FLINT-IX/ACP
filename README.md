# What is ACP?
ACP stands for ACE Content Ported. The goal of this addon was to bring over some of the ACE missiles that i felt could fit into the curret ACF 3 missile lineup well enough. The handful of missiles added are balanced to best keep their own sense of identity while still leaving reasons to use base addon ACF 3 Missiles.

## FAQ
- Best place for feedback?
- Discord, I suck with github.

- Can I disable the tactical weapons?
- Yes you can! In the server side settings of the acf menu, acfm_tac can be set to 0.

- What does the future of this addon look like?
- More or less the addon is done, the only future changes should be if a game update breaks anything, ACF updates, or balance (Will be the main forcus now that the addon is released).
